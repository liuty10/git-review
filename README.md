# git-review

## git tag
To view existing tags:
$ git tag
v0.1
v1.3

To create a tag:
$ git tag -a v1.4 -m 'my version 1.4'
$ git tag
v0.1
v1.3
v1.4

Show information about v1.4
$ git show v1.4

## git branching

Check all existing branch
$ git branch

Create a new branch
$ git branch dev

Switch to dev branch
$ git checkout dev
